test.sh
test.sh 는 shell프로그램으로 만들어졌으며, 알고리즘 3차 과제 test를 위해 만들어졌습니다.
모든 input txt 파일에 대해 알고리즘 별 테스트를 실행해줍니다.
테스트 실행 후, 각 input파일에 대해 .zz 파일과 .zz.yy 파일이 생성됩니다.


[사용법]
	vi test.sh 로 편집기를 열어준다.
	8번째, 19번째 줄에 [Binary File] 부분을 본인의 binary 실행파일 명으로 바꿔준다.
	저장 후, 다음 명령어를 통해 테스트를 시작한다.

	sh test.sh


[주의사항]
1) 테스트 실행 시, 기존 .zz 파일과 .zz.yy 파일은 모두 사라집니다.

2) input 파일은 무조건 _input.txt 로 끝나야 합니다.

check.sh
check.sh 는 shell 프로그램으로 만들어졌으며, 알고리즘 3차 과제 test의 결과 .zz.yy 파일과 기존 input 파일의 비교를 위해 만들어졌습니다.
우선, make를 통해 체크용 실행파일을 생성해줍니다.
생성된 실행파일을 통해 모든 input 파일과 .zz.yy 파일을 비교하여 같은지 체크합니다.


[사용법]
	sh check.sh
	
	같을 시, Success ~~~ 문구가 뜸
	다를 시, Fail ~~~ 문구가 뜸


[주의사항]
1) 모든 input 파일은 무조건 _input.txt 로 시작해야합니다.
