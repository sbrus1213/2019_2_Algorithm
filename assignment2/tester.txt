tester.sh 는 shell프로그램으로 만들어졌으며, 2차 과제 test를 위해 만들어졌습니다.
모든 input txt 파일에 대해 알고리즘 별 테스트를 실행해줍니다.
테스트 실행 후, 각 input파일에 대해 구현해 놓은 result txt 파일이 생성됩니다.

사용법
	vi tester.sh 로 편집기를 열어준다.
	10번째 줄에 [Binary File] 부분을 본인의 binary 실행파일 명으로 바꿔준다.
	저장 후, 다음 명령어를 통해 테스트를 시작한다.

	sh tester.sh [@Algorithm_index]
	ex. sh tester.sh 2 3
		-> 2, 3 번 알고리즘에 대해 현재 디렉토리의 ~~_input.txt 파일들을 모두 테스트 후, result_ file을 생성해준다.

주의사항
1) 테스트 실행 시, 기존 result 파일은 모두 사라집니다.

2) input 파일은 무조건 _input.txt 로 끝나야 합니다.

3) output 파일은 무조건 result 로 시작해야 합니다.
